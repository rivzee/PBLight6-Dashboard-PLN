Debug Information:

Troubleshooting:

  • Badge tidak muncul: Pastikan ada data realisasi yang belum diverifikasi atau target yang belum disetujui
  • Badge muncul tapi salah angka: Periksa query database untuk menghitung notifikasi
  • Badge tertutup elemen lain: Periksa z-index CSS
  • User bukan asisten_manager: Badge hanya muncul untuk role asisten_manager
Untuk mengecek di dashboard asli:
1. Buka browser Developer Tools (F12)
2. Lihat Console untuk debug information
3. Periksa apakah ada error atau warning
4. Pastikan role user adalah 'asisten_manager'
